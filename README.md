# PlantCV GFP Tutorial

This tutorial describes how to visualize GFP fluorescence and isolate plant tissue from fluorescence images. It uses Phenovation PlantExplorer .INF files to extract GFP, autofluorescence, dark-reference, and chlorophyll frames. The chlorophyll image is used to segment a single plant, and the masked GFP signal is displayed as a pseudocolor image. The workflow can be adapted to other compatible PlantExplorer GFP datasets.

The example setaria image contains four wildtype plants (left side) and four GFP plants (right side). Credit Dong-Yeon Lee from Nusinow lab.
